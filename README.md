# Bakery-part3-website
                                                                                                                                # Sweet Treats Bakery WebProject Overview
This project is a multi-page bakery website developed to promote Sweet Treats Bakery online. The website aims to showcase products, improve customer interaction, and make it easier for customers to learn about services and place enquiries.

Website Goals and Objectives
• Sell bakery products online
• Increase sales through internet visibility
• Reach more customers beyond the physical store
• Make shopping easy and convenient
• Improve customer engagement and communication

Features
• Multi-page website structure
• Consistent navigation across all pages
• External CSS styling
• Responsive card-based layout
• Product gallery
• Hover effects on buttons, links, and cards
• Contact form validation with JavaScript
• Automatic footer year update
• Back-to-top button
• Welcome message on page load

Technologies Used
• HTML5
• CSS3
• JavaScript

JavaScript Enhancements
JavaScript was added to improve interactivity and usability. It displays a welcome message when the site loads, enables the back-to-top button, updates the footer year automatically, and validates the contact form before submission.
